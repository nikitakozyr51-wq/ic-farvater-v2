==========================================================
  ПАПКА ДЛЯ ЗАМЕНЫ ФОТОГРАФИЙ ТОВАРОВ
==========================================================

Здесь лежат КОПИИ всех текущих фото каталога, разложенные по папкам
и названные понятно. Чтобы заменить фото — просто положи свой файл
С ТЕМ ЖЕ ИМЕНЕМ поверх старого, и напиши мне в чате "готово".

----------------------------------------------------------
СТРУКТУРА ПАПОК
----------------------------------------------------------
1-транзисторы\     — по одному файлу на товар, имя = артикул
                     (ET1008.webp, BLF574.webp, D1028UK.png ...)
                     37 файлов = 37 товаров.

2-микросхемы\      — по одному файлу на товар, имя = артикул
                     (ET1310PN1U.webp, 1310ПН1У.webp ...)
                     Латиница (ET...) и кириллица (Milandr) — РАЗНЫЕ товары.

3-преобразователи\ — фото по ВАРИАНТАМ (одно фото = много позиций):
                     irtysh-full-brick / half-brick / quarter-brick — по корпусу;
                     enisey-20 ... enisey-1200, kama-50 ... kama-1000,
                     volga-50 ... volga-2000 — по мощности.

4-конденсаторы\    — arc-case-2 ... arc-case-14 — фото по исполнению (ёмкости).

5-разъёмы\         — фото по типу (вилка / розетка / серия).

----------------------------------------------------------
КАК ЗАМЕНИТЬ ФОТО
----------------------------------------------------------
1. Зайди в нужную папку, найди файл с артикулом товара.
2. Положи свой файл ПОВЕРХ, сохранив то же имя (расширение
   jpg/png/webp не важно).
3. Напиши мне "готово" (или какие именно менял) — я покажу таблицу
   сопоставления и после твоего ОК залью в CMS. Робот подхватит за ~10 мин.

Фон желательно светло-серый (как на карточках сайта), форма — квадрат.

----------------------------------------------------------
ВАЖНО про общие фото
----------------------------------------------------------
- В папке "3-преобразователи" одно фото используется многими позициями
  (напр. volga-700 — все преобразователи ВОЛГА на 700 Вт). Заменишь один
  файл — обновятся все позиции этой группы.
- В "1-транзисторы" 21 товар сейчас на одной серой заглушке LDMOS
  (файлы визуально одинаковые). Ты говорила, что улучшишь их и пришлёшь
  — заменяй по одному, кладя свой файл с нужным артикулом.
- Часть фото микросхем сейчас мелкие технические чертежи распиновки
  (маленький размер файла) — их логичнее всего заменить реальными фото.

Карта "какое фото — сколько позиций" лежит в файле _карта-фото.json
